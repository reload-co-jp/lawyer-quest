# 国会——国権の最高機関と立法権

## 国会の地位（憲法41条）

> **日本国憲法 第41条**
> 国会は、国権の最高機関であって、国の唯一の立法機関である。
>
> — e-Gov 法令検索（https://laws.e-gov.go.jp/law/321CONSTITUTION）

**国権の最高機関**：政治的美称説（通説）。三権分立上の序列を示すものではなく、国民の代表機関として最高の民主的正統性を持つという意味。

**唯一の立法機関**：

- **国会中心立法の原則**：国会以外の機関は立法不可（例外：内閣の政令・最高裁の規則）
- **国会単独立法の原則**：他の機関の参与なしに立法（例外：地方特別法の住民投票・95条）

## 両院制

| 事項     | 衆議院          | 参議院                   |
| -------- | --------------- | ------------------------ |
| 任期     | 4年（解散あり） | 6年（3年ごとに半数改選） |
| 被選挙権 | 25歳以上        | 30歳以上                 |
| 定数     | 465人           | 248人                    |

## 衆議院の優越

| 事項                         | 衆議院の優越                                                      |
| ---------------------------- | ----------------------------------------------------------------- |
| 予算の先議権                 | 衆議院が先議（60条1項）                                           |
| 予算・条約・内閣総理大臣指名 | 両院協議会後も不一致→衆院議決が国会議決（60条2項・61条・67条2項） |
| 法律案                       | 参院否決後、衆院の3分の2以上で再可決→成立（59条2項）              |
| 内閣不信任                   | 衆議院のみ（69条）                                                |

## 国会の種類

| 種類             | 召集                        | 期間  | 内容               |
| ---------------- | --------------------------- | ----- | ------------------ |
| 常会（通常国会） | 毎年1月                     | 150日 | 予算審議等         |
| 臨時会           | 内閣が決定 or 総議員1/4要求 | 不定  | 臨時の案件         |
| 特別会           | 解散後総選挙後30日以内      | 不定  | 内閣総理大臣指名等 |
| 参議院の緊急集会 | 衆院解散中・緊急の必要      | —     | 衆院の職を代行     |

## 議院の権能

各議院が独自に行使できる権能：

- **国政調査権**（62条）
- **資格争訟の裁判**（55条）
- **役員選任・議院規則制定**（58条）
- **議員の懲罰**（58条2項）
- **秘密会の開催**（57条1項ただし書）

## 法律の制定手続

1. 議案の発議（内閣 or 議員）
2. 委員会審査
3. 本会議議決（過半数）
4. 他院送付→同様の手続
5. 天皇の公布（7条1号）
