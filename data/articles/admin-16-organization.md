# 行政組織法——行政機関の種類と権限の委任

## 行政機関の種類

国の行政機関は国家行政組織法・内閣府設置法等で規定。地方は地方自治法が基本。

| 機関     | 内容                                                       |
| -------- | ---------------------------------------------------------- |
| 省       | 内閣の統轄下の最上位行政機関（総務省・法務省等）           |
| 庁       | 省の外局（国税庁・消費者庁等）                             |
| 委員会   | 独立性の高い合議制機関（公正取引委員会・国家公安委員会等） |
| 審議会等 | 諮問・調査機関                                             |

## 行政庁・補助機関・諮問機関

| 機関の類型 | 内容                                                       |
| ---------- | ---------------------------------------------------------- |
| 行政庁     | 行政主体の意思を決定・外部に表示する権限を持つ機関         |
| 補助機関   | 行政庁の職務を補助（副大臣・大臣政務官・事務次官等）       |
| 諮問機関   | 行政庁の諮問に応じて答申・意見を述べる（審議会等）         |
| 参与機関   | 行政庁の意思決定に参与し議決で拘束する（電波監理審議会等） |
| 執行機関   | 行政目的を実現するため実力行使する（警察官・徴税機関等）   |

## 権限の委任と専決（代決）

### 権限の委任

行政庁が自己の権限の一部を他の行政機関に移す。

- 委任された機関が**自己の名と責任**で権限行使
- 委任には法律上の根拠が必要
- 委任元は委任した権限を自ら行使できない（権限の移転）

### 専決（代決）

権限は移転せず、補助機関が行政庁の名で処理する内部的行為。

- 法律上の根拠不要（内部委任）
- 対外的には行政庁の行為として効力発生

## 行政機関相互の関係

### 指揮監督関係

上級行政機関は下級行政機関を指揮監督できる。具体的には：

- 訓令・通達による指導
- 報告・検査権
- 取消・停止権（法令上の根拠がある場合）

### 権限の代理

行政庁の権限を他の行政機関が代理して行使する。

| 区分     | 内容                                     |
| -------- | ---------------------------------------- |
| 授権代理 | 法令の根拠に基づく代理（本来の代理）     |
| 法定代理 | 欠缺・事故の場合に法律上当然に生じる代理 |

## 地方公共団体の執行機関

地方自治法は長（知事・市長等）のほか、複数の執行機関を設置。

| 機関         | 内容                                   |
| ------------ | -------------------------------------- |
| 長           | 都道府県知事・市町村長（総合執行機関） |
| 委員会・委員 | 教育委員会・選挙管理委員会・監査委員等 |

委員会は**独立性**を持ち、長の指揮監督に服しない。
